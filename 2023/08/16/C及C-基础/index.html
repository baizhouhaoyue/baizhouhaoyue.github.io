11.字符串变量

字符串常量 是由双引号括起来的字符序列（字符数组），‘\0’代表一个字符串的结束，隐式显示。

每个字符串的结尾，编译器会自动的添加一个结束标志位’\0’， 即“a”包含两个字符‘a’和’\0’。

13.有符号类型数据与无符号类型数据

1.我们在定义一个变量的时候如果没加任何修饰的话，该变量默认是有符号类型,

2.如果要定义一个无符号类型的数据，一定要显式定义，通过unsigned来定义。

char chtmp；//定义一个有符号的char类型数据，

unsigned char chtmp；//定义一个无符号的char类型数据。

有符号数：

数据二进制的最高位为符号位 其他位为数据位。 最高位为1 表示负数 最高位为0 表示正数

以一字节为例：xddd dddd 1111 11111000 00000000 0000~0111 1111 -127 ~-0 ~+0 ~+127

将-0看成-128 有符号的单字节表示的数据范围是-128~127。

无符号数：

没有符号位 所有二进制位都是数据位，以单字节为例，它所能表示的数据范围为：0000 0000 ~1111 1111 ，0 ~ 255

15.原码、反码、补码

注意：在计算机内部所有的数据都是以补码的形式存储

无符号数：

补码==反码==原码

有符号正数：

补码==反码==原码

有符号负数：

先得到反码：除符号位不变，其它位取反，

补码 = 反码 + 1

负数在计算机以补码的方式存储

非负数在计算机以原码的方式存储（非负数的原码==反码==补码）

对数据的存：

八进制及十六进制都是以原码方式进行存储。

负数在计算机以补码的方式存储。

非负数在计算机以原码的方式存储（非负数的原码==反码==补码）。

对数据的取：

对数据的取如果是对 无符号变量 进行取值，输出内存的原样数据。

如果是对 有符号变量 进行取值，系统会去看内存的最高位，如果最高位为0 表明正数， 内存原样 输出。

系统会去看内存的最高位，如果最高位为1 表明负数， 将内存数据求补码（得到原码） 输出。

16.其他关键字

16.1 const关键字：只读变量

初始化时是不会分配内存空间。

const int itmp = 10; int *p = (int *)&itmp; //开辟内存空间 *p = 1000; cout<< “*p = “ << *p << “ itmp = “ << itmp << endl;

16.2 register修饰寄存器变量

高频率使用 的变量建议定义为寄存器变量， 目的：提高访问效率

定义一个register变量：register 类型名 变量名

在c++中即使声明了register变量，但它依然存储在内存中：

register int ntmp = 10; cout << “ntmp地址 = “ << &ntmp << endl;

16.3volatile 关键字

防止编译器对该变量进行优化，强制访问内存。

volatile int data=0;//对data的访问 必须从内存访问
