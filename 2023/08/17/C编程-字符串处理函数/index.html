C编程____字符串处理函数

第一章 strlen测量字符串的长度

函数定义：size_t strlen(const char *); ///< 防止用户修改该参数

函数功能：获取字符串的长度，但不包括‘\0’。注意与sizeof的区别

头文件： #include <string.h>

第二章 字符串拷贝函数strcpy，strncpy

函数定义： char * strcpy（char * dst， const char * src）;

函数功能：实现字符串的拷贝

函数定义：char *strncpy(char *dest, const char *src, size_t n);

函数功能：将 src 指向的字符串前 n 个字节，拷贝到 dest 指向的内存中

注意：

1、 strncpy 不拷贝 ‘\0’

2、如果 n 大于 src 指向的字符串中的字符个数，则在 dest 后面填充 n-strlen(src)个 ’\0’

第三章字符串追加函数strcat、strncat

函数定义： char *strcat(char *dest, const char *src);

函数功能：strcat 函数追加 src 字符串到 dest 指向的字符串的后面。追加的时候会追加’\0’。

函数定义：char *strncat(char *dest, const char *src, size_t n);

函数功能：strncat函数追加src字符串的n个字节到dest 指向的字符串的后面。追加的时候会追加’\0’。

注意：如果 n 大于 src 的字符个数，则只将 src 字符串追加到 dest 指向的字符串的后面追加的时候会追加’\0’

注意：字符串dest的空间足够充足。

第四章 字符串比较函数strcmp、strncmp

函数定义：int strcmp(const char *s1, const char *s2);

函数说明：逐个比较s1与s2字符ascII 码，一旦比较出大小返回。

函数定义：int strncmp(const char *s1, const char *s2, size_t n);

函数说明：比较 s1 和 s2 指向的字符串中的前 n 个字符

返回值：

如果 s1 指向的字符串大于 s2 指向的字符串 返回 大于0

如果 s1 指向的字符串小于 s2 指向的字符串 返回 小于0

如果相等的话返回 0

第五章 字符串中查找字符strchr、strrchr

函数定义： char *strchr(const char *s, int c);

函数说明：在字符指针 s 指向的字符串中，找 ascii 码为 c 的字符，返回首次匹配字符c的地址，如未找到返回NULL

函数定义： char *strrchr(const char *s, int c);

函数的说明：末次匹配，找最后一次出现的 ASCII 为 c 的字符，并返回最后出现字符c的地址，如未找到返回NULL

第六章 字符串匹配函数strstr

函数说明：char *strstr(const char *haystack, const char *needle);

在 haystack 指向的字符串中查找 needle 指向的字符串，也是首次匹配。

返回值：

找到了：找到的字符串的首地址

没找到：返回 NULL
